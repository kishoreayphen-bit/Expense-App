## âš ï¸ **IMPORTANT NOTES**

### **1. You MUST Select Users First**
- The "CREATE GROUP" button is **disabled** until you select at least one user
- Go to "Select Users" section first
- Tap "Load Users"
- Check at least one user
- Then the button will be enabled

### **2. Groups = Teams**
- In your app, "Groups" and "Teams" are the same thing
- Create in **Splits tab**
- View in **Groups tab**

### **3. Why in Splits Tab?**
- Groups/Teams are designed for splitting expenses
- You create a group with members
- Then you can split expenses among those members
- That's why the creation UI is in the Splits screen

## ğŸ” **TROUBLESHOOTING**

### **Problem: "I don't see the Splits tab"**
**Solution:** Check your bottom navigation bar. It should be there between FX and Groups tabs.

### **Problem: "CREATE GROUP button is disabled/grayed out"**
**Solution:** You need to select at least one user first:
1. Scroll up to "Select Users"
2. Tap "Load Users"
3. Check at least one user
4. Scroll back down
5. Button will be enabled

### **Problem: "Load Users shows no users"**
**Solution:** 
- Try searching for a specific name or email
- Make sure you have company members
- The app might show demo users if no real users exist

### **Problem: "I created a group but don't see it"**
**Solution:** 
- Go to the **Groups tab** (bottom navigation)
- Your new group should appear there
- Pull down to refresh if needed
